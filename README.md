# Valkie - Your Personal Voice Assistant

Valkie is an open-source, customizable, and powerful voice assistant created by **PDXF** and the team at **VALK**. Valkie helps you interact with your system using natural language commands and can perform various tasks such as telling jokes, managing time, opening websites, checking system information, and more!

---

## Features

- **Tell jokes**: Valkie can tell jokes, including time-related, number-based, and more.
- **Time and Date**: Ask Valkie for the current time or date.
- **Sound and Media**: Valkie can play sounds or open websites for you.
- **Fun Facts**: Request a random fun fact.
- **System Information**: Get details about memory, battery, or perform calculations.
- **Search the Web**: Valkie can search Google for anything you need.
- **Text Editor**: Open your favorite text editor directly from Valkie.
- **Voice Control**: Activate listening mode and interact with Valkie hands-free.

---

## Installation

### Prerequisites

Ensure you have the following installed on your machine:

- Python 3.x
- Pip
- SpeechRecognition (for voice commands)
- Pyttsx3 (for text-to-speech functionality)
- Additional Python libraries for specific tasks (listed below)

### Install Dependencies

Clone the repository and install the required dependencies:

```bash
git clone https://github.com/PDXF/Valkie-Assistant.git
cd Valkie
```

---

## How to Use

After setting up Valkie, simply run the script or execute the assistant through a terminal. You can use voice commands or type text commands, and Valkie will respond accordingly.

### Example:

```bash
python valkie.py
```

## Available Commands

Valkie can perform the following tasks based on the commands you provide:

### Jokes:
- **Tell a joke**: `tell_joke()`
- **Tell a time joke**: `valkie_joke()`
- **Tell a number joke**: `number_joke()`

### Time and Date:
- **What time is it?**: `tell_time()`
- **What's the date?**: `tell_date()`

### Fun and Media:
- **Play sound**: `play_sound()`
- **Open website**: `open_website()`
- **Perform a calculation**: `perform_calculation()`
- **Tell me a quote**: `tell_quote()`
- **Tell me a fun fact**: `tell_fun_fact()`
- **Give me a fun fact**: `tell_fun_fact()`

### System Information:
- **Check memory**: `check_memory()`
- **Check battery**: `check_battery()`
- **Shut down**: `shut_down()`

### Text Editor:
- **Open text editor**: `open_text_editor()`
- **Open notepad**: `open_text_editor()`
- **Start notepad**: `open_text_editor()`

### Search:
- **Search Google**: `search_google()`
- **Search on Google**: `search_google()`

### Voice Commands:
- **Unmute listening**: `unmute_listening()`
- **Turn on listening**: `unmute_listening()`
- **Activate listening**: `unmute_listening()`
- **Start listening**: `unmute_listening()`
- **Valkie, listen now**: `unmute_listening()`
- **Wake up Valkie**: `unmute_listening()`

### Sleep Mode:
- **Go to sleep**: `sleep()`
- **Sleep now**: `sleep()`
